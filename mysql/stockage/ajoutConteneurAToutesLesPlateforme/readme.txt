Ajout de certains conteneurs à toutes les collections d'une plateforme qui ne l'ont pas déjà.

Renseigner dans les 2 fichiers sql les noms des conteneurs concernés ainsi que celui de la plateforme impactée

le script 00-SELECT--A_ADAPTER....sql permet de visualiser les données qui seront insérées pour éventuellement faire des contrôles et garder une trace
le script 01-INSERT--A_ADAPTER....sql permet de lancer les insertions.
